Basic Information

This page includes every RetroStudio game that I've made. Most of them don't really matter that too much.

birdmartinyt's testing place

This place used to be a recreation of my Roblox game called birdmartinyt's Land.
However, I later replaced it with a DavidBlocks 2000 theme recreation but that project got scrapped.
Nowadays, I use that place to test things out with RetroStudio.

Fencing [Retro Blox Edition]

This is a recreation of a Roblox game called Fencing made by StickMasterLuke.
Which funny enough, the creator of that original game actually liked my twitter posts about this RetroStudio recreation!
This recreation has around 90 visits.

Cry about it

This is a shitpost recreation of the 2009 Happy Home of Robloxia, but using a Mid 2015 theme on purpose.
The game was made because of an official Retro Dev Tweet saying "For every game on RetroStudio that uses Mid 2015 with a -2013 map we will slaughter 1 small animal."

BADBLOX (RETROBLOX BUT BAD)

My second favourite RetroStudio game that I've ever made.

BADBLOX is purposefully a poor recreation of RETROBLOX.
This game uses an alt account called "RetroDev_Remakes" to remake RETROBLOX games on RetroStudio. The reason why I made the game remakes on a separate alt account, is because I didn't want to fill up my profile with those said games.

The game has around 453 visits.
Although due to multiple Roblox and RetroStudio updates, the game may be slightly broken.

The following games BADBLOX has are:
TEST - This only exists to test out the teleporting system.
David Blocks 1994 - This used to be a Crossroads map with a 1994 theme, but RetroStudio REPLACED the theme with late 2015 theme instead.
NPC Dialog Demo - This is a recreation of a game made by Shedletsky. In this game, you can test out the dialog system.
Sword Fight on the Heights Original - This is a recreation of a Roblox sword fighting game made by Shedletsky.
RETROBLOX Skate Park - This is a unfinished recreation of the RETROBLOX skate park game made by Retro Dev.
Happy Home in Robloxia - This is a recreation of a 2009 starter place. This was copied from the "Cry About It" map lmao.
