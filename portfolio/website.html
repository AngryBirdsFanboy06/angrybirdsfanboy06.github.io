Basic Information

This website was mainly made for sharing all things related to myself, projects and more in one place. Along with other domain related stuff, such as the Bluesky accounts.

The website style is heavily based on the "Windows Classic Theme" (More specifically the Windows 2000/XP/Vista/7 version).
The reason why I picked that as my style for this website is because of it being a big part of my childhood.

Oh, and not to mention that using this type of style for a website, makes it so I don't have to purposefully worry about the website looking weird on mobile lmao.

Development

Around March 24th to April 2nd, 2026. I made a website layout mockup via Photoshop for the preparation on actually coding the website, including the style itself, as well as to make sure I don't get lost. This was VERY helpful.

The development in regards to completing the main framework, style, and main pages only took around 15 days to complete. In those days, I managed to complete 9 pages, as well as learning HTML and css.

Although at the time, side pages for Socials and Portfolio pages weren't made just yet, along with other minor features such as: the website buttons iframe page still being unfinished, drawings being temporary unfinished, no JavaScript stuff, domain related stuff and more.

From April 6th to May 3rd, 2026. I finally finished all of the socials and portfolio pages for my website. A long with adding a robots.txt file.
