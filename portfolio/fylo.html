Basic Information

Fylo is a server security-heavy and moderation focus Discord bot created by ojaqob and JoshAmour, It's purpose is to keep Discord servers safe and protected with features such as automated server backups, detailed logging, and much more for completely free.

My Involvement

Was involved during the beta testing of the bot, as well as using the bot early on Retro Dev Wiki Discord server before it got added on Retro Dev itself.
Was also involved with suggesting multiple features and giving feedback.
