Basic Information

OS Menu is a custom menu based off the "XBOX 360"'s Xbox Guide (NXE version). The Roblox version of this was supposed to be an updated custom menu for The OS Simulation Project's Roblox games.

However on October 3rd 2024, all of The OS Simulation Project Roblox games (including OS Menu itself) got privated from Roblox, due to one of their games (Xbox 360 GUI Test) getting taken down by Microsoft itself, as well as not being satisfied with the direction Roblox was going.

Nowadays, OS Menu has been ported to the Unity Game Engine as a way to learn on how to use the game engine itself.
Along with OS Menu having a JavaScript version made for the official "The OS Simulation Project" website.

All of The OS Simulation Project Roblox games are available, both on the official Github page and the Internet Archive.
